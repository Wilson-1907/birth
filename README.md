# Birthday Website

## Open it

- Open `index.html` in your browser (double-click it), or
- In VS Code/Cursor, right-click `index.html` → “Open with Live Server” (if you have it).

## Customize

Use the **Settings** section on the page to set:
- The birthday person’s name
- The birthday date
- A message + “from” signature

Settings are saved locally in your browser.

## Share

Click **Copy share link** to generate a URL with the current settings.
